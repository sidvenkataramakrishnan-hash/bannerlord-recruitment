Some new ways to hire troops. So far we have:


-HireablePartyBehavior - Add in new culture specific troops hireable piecemeal from towns


-MercenaryPartyBehavior - Pick up a visiting mercenary party. The parties are designed to be randomised; not all taverns will have a party visiting.  You can fire your mercenary party at any tavern.

-MercenaryPatrolBehavior - Allows towns to hire mercenaries. Will hire at three towns each time the event fires, adding more variety to patrols.
